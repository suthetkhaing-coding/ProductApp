# ProductApp (ASP.NET Core + Dapper)

## Features
- List, Add, Edit, Delete Products
- MSSQL + Dapper ORM

## How to Run

1. Clone this repo
2. Update `appsettings.json` with your DB connection string
3. Run the SQL script to create the `Product` table
4. Run the app with `dotnet run`

## Technologies
- ASP.NET Core MVC
- Dapper
